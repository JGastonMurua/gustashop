# 🛍️ GustaShop

**GustaShop** es un e-commerce ficticio desarrollado como entrega final para el curso de React. Permite explorar productos, agregarlos al carrito, ajustar cantidades y gestionar el listado desde un panel lateral tipo offcanvas, con diseño responsive y experiencia moderna.

---

## ✨ Características

- 🔍 Buscador en tiempo real
- 🛒 Carrito con agregado, eliminación y modificación de cantidades
- 🎨 Diseño 100% responsive con tema uva personalizado
- 🧠 Estado global con React Context API
- 📦 Consumo de API externa ([FakeStoreAPI](https://fakestoreapi.com/))
- 🚀 Interfaz profesional tipo e-commerce real

---


## ⚙️ Tecnologías usadas

- React
- Vite
- Bootstrap 5
- React Context API
- SweetAlert2
- HTML5 + CSS3
- JavaScript moderno (ES6+)

---

## 🛠 Instalación local

1. Cloná este repositorio:

git clone https://github.com/JGastonMurua/gustashop.git
cd gustashop
Instalá las dependencias:
npm install bootstrap sweetalert2 react-router-dom

Ejecutá la app:
npm run dev
Abrí tu navegador en http://localhost:5173.
